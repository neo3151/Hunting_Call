## Quick Steps

1. **Main Store Listing** → Paste app name, descriptions
2. **Graphics** → Upload app icon and feature graphic
3. **Contact Details** → Enter email and privacy policy URL
4. **Categorization** → Select "Sports" category
5. **Screenshots** → Upload when ready
6. **Save** → Save the listing

All files are in the repository at `assets/play_store/`
